# GrowthHacker

#### 分享主题
1. 分享主题不能脱离产品技术
2. 选题方式    
   - 优先个人指定选题
   - 个人选题需要团队投票确认
   - 团队指定选题
3. 鼓励邀请外部人员分享
4. 鼓励分享和实际应用相结合的主题
5. 选题类型：    
    - 语言基础及主题    
    - 编程思想及编程技巧    
    - 实际问题及解决方案    

#### 分享要求
1. 使用脑图梳理知识点
2. 配以示例demo
3. 扩展学习建议
